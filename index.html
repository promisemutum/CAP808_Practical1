Attributes in 'img' tag in HTML, are additional properties that you can set to control how the image behaves and is displayed on a webpage. These attributes provide specific instructions to the browser, such as where to find the image file (src), what text to display if the image fails to load (alt), the dimensions of the image (width, height), and other aspects like how the image should be loaded (loading)

Q2. Compare HTML with HTML5.

HTML and HTML5 are both versions of the HyperText Markup Language, which is used to structure and present content on the web. HTML5 is the latest version, introducing new features like native audio/video support, semantic elements, and offline storage. HTML5 is the latest version of HTML and comes with several improvements and new features.

Key Differences:

HTML HTML5
Not mobile-friendly. Optimized for mobile devices.
No native audio/video support. Supports audio and video tags.
Uses cookies for storage. Uses localStorage and application cache.
JavaScript runs only in the browser. Supports JS Web Workers.
Uses external technologies for vector graphics. Supports SVG and Canvas.
No drag-and-drop functionality. Supports drag-and-drop.
Complex doctype and character encoding. Simple doctype and encoding.
Requires third-party libraries or plugins to create graphics. Includes the canvas element for drawing graphics on the fly and supports
Scalable Vector Graphics (SVG) directly in the markup.
